# Hospital Emergency Room Dashboard

## Project Overview
This project is an interactive Excel dashboard designed to analyze hospital emergency room data.

## Features
- Monthly report analysis
- Patient satisfaction score
- Patient waiting time analysis
- Admission status tracking
- Gender distribution
- Age group analysis
- Department-wise patient analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
